bcp-example-1
=============

Standalone version of <https://github.com/district10/cmake-templates/tree/master/boost>.

See my blog post <http://tangzx.qiniudn.com/post-0110-boost.html> for detail.

How to?

1.  download or git clone this repo
2.  unzip include.zip (you can use `cmake -E tar xJ include.zip` in command-line)
3.  cmake
4.  make
